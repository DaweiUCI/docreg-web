Help FAQ

I can't login...

Please try your GNET account username, e.g. "sabernethy". If this fails please send an email to docreg@aviatnet.com

I can't download documents, or downloads are very slow...

You can change the download server on the user profile preferences page. Please select the DocReg server closest to your location.

Can I edit documents with this service?

Yes. You can try editing and submitting a test document to familiarise yourself with this feature.

  • Document Register Test - SC
  • Document Register Test - NZ

Can I create new documents with this service?

Yes. Use the create document feature on the menu bar.

I wish this service would do X, or would not do Y...

Great! Please add your feature ideas, defect reports, or general comments to the feedback wiki page or email docreg@aviatnet.com.

What browser should I be using?

For best results we recommend you use a recent version of Google Chrome or Mozila Firefox. Use of Internet Explorer may result in problems...
